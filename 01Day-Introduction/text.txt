HTTP response status codes
HTTP response status codes indicate whether a specific HTTP request has been successfully completed. Responses are grouped in five classes:

1xx=>Informational responses (100 – 199)=>Hold on=> Processing Phase
2xx=>Successful responses (200 – 299)=>Here you Go
3xx=>Redirection messages (300 – 399)=>Go Away
4xx=>Client error responses (400 – 499)=>404=>Page Not Found=>You Screwed Up
5xx=>Server error responses (500 – 599)=>Bad Gateway/ Not able to handle => I screwed up

https://developer.mozilla.org/en-US/docs/Web/HTTP/Status
